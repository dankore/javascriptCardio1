Array Methods - Filter, Reduce and Sort

Check your browser console for solutions. Exercise created by Wes Bos.

Filtering Arrays

  • Creates a new array
  • If no element of array is filtered, an empty array is returned
  • var newArray = arr.filter(callback(element[, index[, array]])[, thisArg])Taken from MDN

Reducing Arrays

  • Condenses an array elements to a single value
  • If array has no element, method doesn't run
  • arr.reduce(callback[, initialValue])Taken from MDN

Sorting Arrays

  • Sorts arrays according to your specs in the compareFunction
  • By default, it sorts an array alphabetically from smallest to biggest
  • doesn't return
  • arr.sort([compareFunction])Taken from MDN
